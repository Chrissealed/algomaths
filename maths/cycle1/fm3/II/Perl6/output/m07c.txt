# ------------------
# C = 13/2 × 2.
-----------------------------------------------------------------------------------
Les fractions 13/3 et 2/1 sont irréductibles.
Signe du produit de 13/3 par 2/1 : +.

On multiplie les numérateurs de chaque fraction puis les dénominateurs :
Facteurs premiers obtenus pour 13 × 2 = 26 : 2 13.
Facteurs obtenus pour 3 × 1 = 3 : 1 3.
Fraction résultante : 26/3.
